The request spent most of the time in reading the request entity body from the client. This can occur in the following cases:
  1. The client is posting a very large request to the server. Check the cs-bytes property of this request, which represents the size of the request content, to understand if that's the case
  2. The Client is behind a slow network (for e.g. a mobile client) or in a different geography or geo-location
  3. There is network latency between the client and this application
